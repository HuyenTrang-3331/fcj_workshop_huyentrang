In this worklog, I present my learning and practice process in the **First Cloud AI Journey** program. The worklog was completed over **12 weeks**, from **April 17 2026 to July 09 2026**.

During the learning process, I successively studied AWS foundational services, practiced labs by topic, and recorded achievements after each week. The worklog content not only focuses on completing modules and labs, but also includes the process of architecture analysis, security checks, cost optimization, log monitoring, error handling, and resource cleanup after practice.

The main contents include: getting acquainted with AWS Cloud, setting up the account and initial security, managing IAM, building networks with VPC, deploying EC2, working with Storage, Database, Data Analytics, Security, Monitoring, Cost Optimization, and synthesizing knowledge at the end of the learning period.

**Week 1:** [Getting started with AWS, AWS Free Tier, AWS Console and AWS CLI](1.1-week1/)

**Week 2:** [Learning AWS Cloud, Global Infrastructure, AWS Budget and AWS Support](1.2-week2/)

**Week 3:** [Learning Amazon VPC, Networking, Hybrid DNS, VPC Peering and Transit Gateway](1.3-week3/)

**Week 4:** [Learning Amazon EC2, AWS Backup, Storage Gateway, S3 Static Website and CloudFront](1.4-week4/)

**Week 5:** [Learning AWS Storage Services, Amazon S3, AWS Backup, Storage Gateway, FSx and CloudFront](1.5-week5/)

**Week 6:** [Learning AWS Security, IAM, KMS, CloudTrail, Security Hub and access management](1.6-week6/)

**Week 7:** [Learning Database on AWS, Amazon RDS, Aurora, Redshift, ElastiCache and Database Migration](1.7-week7/)

**Week 8:** [Learning Data Analytics on AWS, DynamoDB, Glue, Athena, QuickSight, Lambda and Redshift](1.8-week8/)

**Week 9:** [Reviewing AWS IAM, account security, cost management and resource optimization](1.9-week9/)

**Week 10:** [Analyzing Web Application Architecture on AWS, CloudFront, WAF, ALB, EC2, S3 and RDS](1.10-week10/)

**Week 11:** [Learning Monitoring, Logging, CloudWatch, CloudTrail, GuardDuty, Security Hub and AWS Config](1.11-week11/)

**Week 12:** [Synthesizing knowledge, reviewing labs, cleaning up resources and completing the report](1.12-week12/)
